# login-simulation
Simulación de login, no tiene seguridad de hecho es facil de pasar, es solo una simulación que permite ver como sería un login estilo macOS.

Es un simple archivo .py diseñado para mi Debian 11 Bullseye, para ejecutarlo simplemente descargalo, ubicate en su directorio correspondiente y ejecutalo con python "python3 .login.py" el comando variará dependiendo de la versión de python o del mismo dispositivo.
